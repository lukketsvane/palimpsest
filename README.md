# Palimpsest

**Diskurs som overflate, infrastruktur som nerver: Ein materialistisk kritikk av Edward Said sitt Orientalism**

This repository contains the manuscript, figures, source materials, and build scripts for an academic paper exploring the tension between discourse analysis and materialist critique. The article uses an artistic project—a palimpsest consisting of Mahmoud Darwish's poem *Tibaq* (2003) inscribed on industrial prism film from a dismantled LCD screen—to investigate and critique Edward Said's *Orientalism* (1978).

## Repository Structure

*   **`build_manuscript.py`**: A Python script to compile and format the manuscript.
*   **`figures/`**: Contains the visual assets, collages, and figures (e.g., `figur_01.png` to `figur_10.png`) referenced in the text.
*   **`sources/`**: An archive of the reference materials, including works by Said, Chomsky, Parenti, Ahmad, and others in PDF and EPUB formats.
*   **`references.bib`**: The BibTeX file containing all citations for the manuscript.
*   **Documents**: Preprints, submission files, and formatting templates (e.g., `PALIMPSEST_preprint_v2.pdf`, `PALIMPSEST_submission.docx`, `formakademisk_template.docx`).

## Abstract Summary

The article argues that Said's *Orientalism* contains a fundamental contradiction between a materialist argument (Orientalism as a consequence of colonialism) and an idealist argument (Orientalism as a cause of colonialism). It asserts that power follows material structures, not merely discourse, and demonstrates this through the concept of "Occidentalism" and an extended reading of Darwish's elegy for Said.
